概述

全局鼠标跟随，可自定义，淦！淦！淦！淦！

示例

根据你的文件位置来链接插件所需要的文件

<script src="cursor.js"></script>
HTML 代码:
<div class="cursor"></div>
CSS 代码:
.cursor {
    width: 50px;
    height: 50px;
    position: fixed;
    z-index: 9999;
    top: 0;
    left: 0;
    transition: opacity .3s, color .4s;
    background-color: black;
    border-radius: 50%;
    opacity: 0;
    pointer-events: none;
    backface-visibility: hidden;
}
body:hover .cursor{
    opacity: 1;
}

#cursor {
    pointer-events: none;
    position: fixed;
    transform-origin: center center;
    mix-blend-mode: difference;
    top: 0;
    left: 0;
    z-index: 1000;
    opacity: 0;
    transition: 0.6s;
}
#cursor span {
    position: absolute;
    display: block;
    width: 26px;
    height: 26px;
    border-radius: 20px;
    background-color: #fff;
    transform-origin: center center;
    transform: translate(-50%, -50%)
}
body:hover #cursor{
    opacity: 1;
