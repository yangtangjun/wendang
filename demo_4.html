概述

自定义分页器，当分页器类型设置为分式时可用。

示例

JS 代码:
pagination: {
    el: '.swiper-pagination',
    clickable:true, //给你的分页器绑定点击事件
    type: 'fraction',
    renderFraction: function (currentClass, totalClass) {
        return '<span class="' + currentClass + '"></span>' +
            '<div class="line"></div>' +
            '<span class="' + totalClass + '"></span>';
    },
    formatFractionCurrent:function (number) {
        return number<10?'0'+number:number;
    },
    formatFractionTotal: function (number) {
        return number<10?'0'+number:number;
    }
},

复制

复制成功

如果想要一个swiper多个分页器或者想把元素传入分页器

可以有n个$('.swiper .item')

$('.swiper .item').each(function(index,ele){
     new Swiper ($(ele).find('.swiper')[0],{
        pagination: {
            el: $(ele).find('.swiper-pagination')[0],
            bulletClass:'paging',
            bulletActiveClass:'active',
            renderBullet: function (index, className) {
                let src=$(ele).find('.swiper-slide').eq(index).find('img').attr('src')
                return  '<div class="'+className+'">'+
                            '<img src="'+src+'" alt="">'+
                        '</div>';
            },
        },
     })
})

如果想要分页器不动，在基础上做出改变

pagination: {
    el: '.swiper-pagination',
    renderBullet: function (index, className) {
        return '<span class="' + className + '">' + '0' + (index + 1) + '</span>';
    },
},
